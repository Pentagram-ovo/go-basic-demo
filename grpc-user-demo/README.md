# gRPC User Service 项目说明
本项目是基于 Protobuf 与 gRPC 实现的用户微服务演示项目，用于学习与理解微服务之间的远程调用方式。

## 项目实现功能
- 提供独立的用户服务，对外暴露远程调用接口
- 实现用户信息创建功能，支持传入用户ID、用户名、密码进行创建
- 实现根据用户ID查询用户信息的功能
- 支持服务端与客户端分离部署，实现跨服务通信
- 采用高效序列化格式传输数据，性能优于传统JSON

## 项目使用技术
- Go 语言开发
- Protobuf 3 作为接口定义语言，用于定义数据结构与服务接口
- gRPC 高性能远程调用框架，实现服务间通信
- 基于 HTTP/2 协议传输

## 项目结构说明
- 项目分为接口定义文件、自动生成代码、gRPC 服务端、gRPC 客户端四部分
- 服务端负责提供用户创建与查询能力
- 客户端负责远程调用服务端接口
- 接口定义文件统一管理数据结构与服务规范

## 学习目标
- 掌握 Protobuf 的基本用法与接口定义方式
- 掌握 gRPC 的服务端与客户端编写流程
- 理解服务间通信的原理与使用场景
- 能够独立完成基础 RPC 服务的设计与实现

## 简历描述
掌握 Protobuf 与 gRPC，能够编写微服务接口并实现服务间通信。
